🔧 トラブルシューティング

よくある質問

Q: HTML ファイルが開かない
A: ブラウザにドラッグ&ドロップ、または「ファイルを開く」から選択してください。
Q: Instagram で診断ツールへのリンクを張りたい
A: プロフィール欄に GitHub Pages URL を入れるか、Linktree などのランディングページサービスを使用してください。
Q: 診断結果をカスタマイズしたい
A: HTML ファイルの JavaScript 部分（resultMessages オブジェクト）を編集してください。
Q: リンクスタイルを変更したい
A: CSS セクション（--color-primary など）の変数値を変更してください。
