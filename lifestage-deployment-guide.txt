## 🚀 デプロイ手順

### STEP 1：ローカルリポジトリを準備

```bash
# 1. ローカルで GitHub リポジトリをクローン
git clone https://github.com/minamina04070-stack/minamina04070-stack.github.io.git
cd minamina04070-stack.github.io

# 2. 新しいフォルダを作成（ツール用）
mkdir lifestage-diagnosis
cd lifestage-diagnosis
```

### STEP 2：ツールファイルをコピー

```
lifestage-diagnosis.html 
→ lifestage-diagnosis/ フォルダ内に「index.html」として保存

※ ファイル名を「index.html」に変更するのが重要です
  （これにより https://minamina04070-stack.github.io/lifestage-diagnosis/ でアクセス可能に）
```

### STEP 3：Git で push

```bash
# リポジトリのルートディレクトリに戻る
cd ..

# ファイルをステージング
git add lifestage-diagnosis/

# コミット
git commit -m "feat: add lifestage diagnosis tool (ライフステージ診断ツール追加)"

# プッシュ
git push origin main
```

### STEP 4：確認

5～10分待つと、以下の URL でアクセス可能になります：

```
https://minamina04070-stack.github.io/lifestage-diagnosis/
```

## 🎨 カスタマイズポイント

### LINE Official Account リンク

```html
<a href="https://lin.ee/SGVlcAV" target="_blank" class="cta-button">LINE で相談する</a>
```

↓

```html

```

### カラースキーム変更
CSS の色コードを変更すれば、全体の雰囲気を変更できます：

```css
/* メインカラー */
#d4a574  → 別の色コードに変更

/* サブカラー */
#8b6f47  → 別の色コードに変更
```

## 🆘 トラブルシューティング

**Q: 404エラーが出る**
→ ファイル名が「index.html」になっているか確認
→ フォルダ構造が正しいか確認（lifestage-diagnosis/index.html）

**Q: ページは表示されるが、動作しない**
→ ブラウザの開発者ツール（F12） → コンソール をチェック
→ エラーメッセージがあれば確認

**Q: LINE リンクをクリックしても開かない**
→ href の URL を確認
→ https://lin.ee/... の形式が正しいか確認
